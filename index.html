iHadoop by debarshri

A Hadoop menu driven installer and cluster configuring tool.

Install and explore...

Dependencies

None

Install

iHadoop is basically a pseudo mode installer for hadoop.Basic idea is to give a menu driven UI for the users.Once the fully distributed mode is done, then installing and configuring hadoop in various machines will be done in matter of minutes. Before you begin You need to set the JAVA_HOME keep your hadoop tar file at /home/user/ To begin with $ /opt/bin/startsh.sh and rest will follow...
